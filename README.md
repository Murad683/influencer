# Murad Mammadov - Luxury Portfolio

A high-end, cinematic portfolio website built with React, TypeScript, and GSAP. This project features smooth scroll storytelling, parallax effects, and a refined minimalist luxury aesthetic.

## ✨ Features

- **Cinematic Experience**: Immersive dark theme with high-end luxury aesthetics.
- **Smooth Scroll**: Integrated with Lenis for a buttery-smooth scrolling experience.
- **Advanced Animations**: Powered by GSAP for complex scroll-triggered effects and parallax transitions.
- **Responsive Design**: Fully optimized for all screen sizes.
- **Glassmorphism & Liquid Metal**: Modern UI elements for a premium feel.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) (Vite)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/) / [Radix UI](https://www.radix-ui.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/)
- **Smooth Scroll**: [Lenis](https://lenis.darkroom.engineering/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: [TanStack Query](https://tanstack.com/query/latest)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm, yarn, pnpm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Murad683/influencer.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Reusable UI components and section layouts.
- `src/pages`: Main page views.
- `src/hooks`: Custom React hooks for state and animation logic.
- `public`: Static assets and global configurations.
